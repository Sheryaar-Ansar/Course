PUBG Overview

When the game got released for Windows, it received rave reviews. Further, upon being released on mobile phones, it caught like wildfire. The craze for this game spread amongst all the age groups.What started as a recreation game has now turned into an addiction. It is severely impacting the lives of the players and also resulting in various crimes. For instance, a boy killed himself due to PUBG mobile game addiction.
